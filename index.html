Workflow, AI & Machine Learning

News to Email - n8n Workflow

Automated daily news digest built with n8n — runs at 8:00 AM, pulls headlines from ESPN, TechCrunch, and Times of India, extracts the top 3 stories per source, and compiles them into a single, formatted email. Implemented using n8n nodes (Schedule trigger, RSS, Merge + Code for parsing/formatting, Gmail OAuth2) and provided as an importable workflow. Ready-to-run automation for staying updated or integrating into larger notification pipelines.

Technologies: n8n, JSON

Biometric Authentication

A complete facial recognition system to securely launch desktop applications using a webcam. Captures face data, trains a recognition model, and authenticates in real-time. Built in modular Python scripts for offline authentication.

Technologies: Python, OpenCV, face_recognition

Smart File Organizer

Organizes photos by matching faces. The first image is used as reference and the folder is traversed to automatically move matching images; unrecognized faces prompt a user decision.

Technologies: Python, OpenCV

Automobile-Data-Analysis

Automobile Data Analysis explores a classic dataset of cars (1970–1982) to uncover trends in fuel efficiency, horsepower, weight, and acceleration. The project combines exploratory data analysis with a K-Nearest Neighbors regression model to predict car acceleration based on key vehicle attributes.

Technologies: Scikit-learn, Matplotlib, Pandas
